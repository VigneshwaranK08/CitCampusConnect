## **Project Name : CampusConnect (for reference)**



### 1\. Core Objectives



* Allow students to create posts (text, image, video optional)



* Build social connections: follow/followers, friends



* Engage via likes, comments, reposts



* Share college events and updates



### 2\. Mandatory Features (MVP)



* User Module



* Sign up / Login (Email \& Password or Google)



* Profile page: picture, bio, events attended



* Follow/Unfollow or Friend requests



* Post Module



* Create post (text + image)



* Like, comment, repost



#### Simple feed: latest posts from friends/following



* Social Module



* Followers \& Following lists



* Notifications (likes, comments, friend requests, events)



#### UX Essentials



* Clean, minimal design



* Responsive layout



* Easy navigation (tabs for Feed, Events, Profile)



* Smooth scrolling \& interactions



Loading placeholders for posts



### 3\. Tech Stack (Beginner-friendly, scalable)



#### Frontend



React (Web) 



* UI Library: Bootstrap



#### Backend



* Node.js + Express (Beginner-friendly, widely used)



* Firebase or MongoDB (NoSQL, easy to start)



* Authentication: Firebase Auth or JWT



#### Storage



Firebase Storage (for images/videos)



Hosting



* Frontend: Netlify / Vercel



* Backend: Render / Heroku (free tiers for beginner projects)
